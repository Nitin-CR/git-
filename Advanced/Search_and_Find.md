* It is basically filtering your commit history
* It can be done based on
* -> by date --before/--after
* -> by message --grep
* -> by author --author
* -> by file -- <filename>
* -> by branch <branch-A>
  
  
  
 Based on date
  
  ```
  git log --after="yyyy-dd-mm"
  ```
  
 Based on the message
  
  ```
  git log --grep="message"
  ```
  
  BAsed on file
  ```
  git log --Filename.ext
  ```
